Contributing & support
======================

Source repository
-----------------

* Code: https://github.com/rajuaman1/QNPy_Latte
* Issues and questions: open a GitHub issue on the repository.

Development install
-------------------

.. code-block:: bash

   git clone https://github.com/<your-account>/QNPy_Latte.git
   cd QNPy_Latte
   pip install -e .

Building these docs locally
---------------------------

.. code-block:: bash

   pip install -r docs/requirements.txt
   cd docs
   make html            # output in docs/build/html/index.html
